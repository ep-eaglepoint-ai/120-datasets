# Form Builder - Next.js Application

Convert vanilla HTML/JavaScript form builder to modern Next.js 14 with TypeScript and Tailwind CSS while maintaining all functionality.



## Run with Docker

### Run tests (before – expected to fail)

```bash
docker compose run --rm tests-before
```

### Run tests (after – expected all pass)

```bash
docker compose run --rm tests-after
```

### Run evaluation (compares both implementations)

```bash
docker compose run --rm evaluation
```
